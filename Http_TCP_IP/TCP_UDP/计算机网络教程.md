# 数据链路层

数据链路层功能：提供介质访问，和链路管理。

3个基本功能： 

1. 封装成帧。  收到物理层比特流后，根据首部和尾部的标记，从收到的比特流中识别帧的开始和结束。
2. 透明传输。  其实就是保证数据完整性，不把和结束符EOF二进制一样的结束符当做结束符。
3. 差错检测。 比特流在传输的时候，可能出现差错，将0传成1 ，叫比特差错。

ppp协议（点对点协议）：
